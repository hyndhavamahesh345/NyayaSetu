# Contributing to NyayaSetu

First off, thank you for considering contributing to NyayaSetu! It's people like you that make NyayaSetu such a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, open a new issue. Include a clear title, a description, and as much relevant information as possible (OS, Python version, steps to reproduce).

### Suggesting Enhancements
* Open a new issue with the tag "enhancement".
* Describe the feature and why it would be useful.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code follows the existing style.

## Style Guide
* Follow PEP 8 for Python code.
* Keep your commits clean and descriptive.
